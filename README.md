# Implementation of CHIP 8 Emulator in Python

This repo contains implementation of CHIP 8 Emulator. It is meant for educational 
and experimental purposes.

## Run

```
git clone https://github.com/minmin2104/chip-8-emulator.git
cd chip-8-emulator
python src/main.py ./roms/MAZE
```
